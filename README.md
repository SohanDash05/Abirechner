# NotenApp

Eine Desktop-App zur Verwaltung und Berechnung von Schulnoten fuer ein Q12-Informatik-Projekt.

## Ziel

Die App soll Schuelerinnen und Schuelern helfen, Noten uebersichtlich zu speichern, Durchschnitte zu berechnen und den aktuellen Leistungsstand pro Fach zu sehen.

## Geplanter Tech-Stack

- Python
- PySide6 fuer die grafische Oberflaeche
- SQLite fuer lokale Datenspeicherung
- PyInstaller fuer eine spaetere Windows-EXE

## Geplante Funktionen

- Faecher anlegen und verwalten
- Noten eintragen
- Gewichtungen fuer muendliche und schriftliche Leistungen
- Durchschnitt pro Fach berechnen
- Gesamtdurchschnitt anzeigen
- Daten lokal speichern
- Export als CSV

## Entwicklung

Das Projekt wird auf macOS entwickelt und soll spaeter auch auf Windows laufen.
